 </span>
</p>
<p class="p3">
 <span class="s1">
  在株洲，有些比较敏感的维权人士，已经“被旅游”了。“至少有二位朋友已经被带走，不能发出任何声音，还有被警告、约谈的。”他说，“在这个很小的城市里面，就是一部分敏感的人群已经被限制了，还随时会有很多人站出来发声。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  郭胜是一名70后，8964时他读小学六年级，他表示，“记忆中隐隐约约知道这个事件，但那个时候的认识，并不知道什么是对错，直到很多年后跟外界了解了信息之后，才认识到这是中共的屠
 </span>
 <span class="s3">
  杀
 </span>
 <span class="s1">
  行为，是中共对中国人民的一种打压。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  2017年的六四，他曾在
 </span>
 <span class="s4">
  株洲
 </span>
 <span class="s1">
  组织了10个朋友在街头拍摄纪念六四的照片，后遭当局抓捕逃往美国。
 </span>
</p>

<p class="p7">
 <span class="s1">
  他说，“其实这几十年过来，人们就是在这种打压下被蒙蔽。但我们这些觉醒的人，要把信息传递出去，而且会做自己力所能及的事情。”
 </span>
</p>
<p class="p7">
 <span class="s1">
  “就像这些朋友，做这些事情都要面临很大的危险，当局会给他们很多压力，包括工作、学习、孩子读书等各方面，他们还是义无反顾地去做。”
 </span>
</p>
<p class="p7">
 <span class="s1">
  1989年的民运学潮中，从北京到全国各地高校的学生、民众有上百万人参与，反官倒、腐败，提出
 </span>
 <span class="s5">
  经济自由化、民主、法治等政治诉求。6月4日凌晨，
  <span class="s6">
   中共
  </span>
  <span class="s6">
   在
  </span>
  <span class="s7">
   天安门广场进行暴力清场，用
  </span>
 </span>
 <span class="s7">
  机枪射
 </span>
 <span class="s9">
  杀
 </span>
 <span class="s7">
  和坦克碾压学生。
 </span>
</p>
<p class="p10">
 <span class="s1">
  而31年来，中共一直在掩盖六四真相，不断迫害民运人士。近年来，越来越多的人们意识到，中共不可能给六四平反，而清算中共、解体中共才是历史的走向。
 </span>
</p>
<p class="p10">
 <span class="s1">
  郭胜说，“中共现在对国内的压力、控制和打压越来越严，迫害这些维权人士和律师，但越是这样，其实也就更加能体现它们离自己崩溃也就更加接近了。越是高压就越是容易爆炸的时候，已经到边缘了。”
 </span>
