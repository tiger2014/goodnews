   22日，根据DailyNK咸镜南道的消息人士透露，在本月初，有一名50多岁女性金某，她不顾政府管制，在咸兴市内圣川江河坝交易美元，当场被逮捕，并进行了公开审判。
   <br>
    <br>
     其实，金某是一名当地很有名的现金交易商，这几年来，一直背着背包四处进行兑换钱币交易，而赚了不少钱。
     <br>
      <br>
       4月中旬，她在沙埔市场偷偷进行美元与朝币的兑换交易之时，刚好被巡逻的保安员发现而遭逮捕。
       <br>
        <br>
         若依照朝鲜政府所公示的国家牌价来换钱，则朝鲜货币的价格太低了，因此一般民众们会与市场上的个体外汇商贩，进行兑换现金。
         <br>
          <br>
           个人之间进行兑换外汇，一直都是朝鲜当局的监视物件。而且最近也突然对于非法兑换加强了打击力度。
           <br/>
           <br/>
           先前，DailyNK曾经引述朝鲜内部消息人士的话报导，平壤当局为了禁止个人之间进行高价及大量外汇交易，所以组织了一个专门打击小组“1118常务”。朝鲜当局在发行国债之际，为了降低国债发行所发生的负面作用，以稳定外汇市场而设立的。
           <br/>
           <br/>
           根据消息人士表示，金某是在交易现场被发现的，当时她为了不遭没收现金，而进行了一些反抗，双方扭打在一起，以致划伤了保安员的脸，因而将事情弄得越来越大。
           <br/>
           <br/>
           据悉，保安员们拉着金某到她家里，并进行搜查，当下气急败坏的金某，突然坐在装钱的25公斤装的袋子上，并拿出汽油桶与打火机，威胁他们说，钱被没收，还不如都烧光了。
           <br/>
           <br/>
           据消息人士指出，在本月初的公开审判会上，金某因为违反禁止个人换钱的方针，并且妨碍公务，及在挂有金日成和
           <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
            金正恩
           </span>
           肖像的住宅威胁放火等罪行，而遭判无期徒刑。
           <br/>
           <br/>
           消息人士称，在进行审判的前一天，才通知审判日程，第二天就召集了咸兴市人民班长、沙埔市场生意人等来进行公开审判。这次遭判处无期徒刑的金某，当场立刻就被戴上手铐，并用车拉走。
