 <p>
  【看中国2020年4月24日讯】
  <strong>
   中共病毒
  </strong>
  （武汉肺炎）疫情造成驱动中国出口成长的
  <strong>
   农民工
  </strong>
  成受创最重阶层，薪资
  <strong>
   收入
  </strong>
  损失严重。
  <br>
   <br>
    咨询机构策纬中国（Trivium China）报告指出，至二月底，在外地有工作的
    <span href="https://zh.wikipedia.org/wiki/%E6%B0%91%E5%B7%A5" target="_blank">
     农民工
    </span>
    下滑近30%、仅约1.225亿人，月薪
    <span href="https://zh.wikipedia.org/wiki/%E6%94%B6%E5%85%A5" target="_blank">
     收入
    </span>
    平均减7.9%至3,680人民币。
    <br>
     <br>
      据《路透社》报导，
      <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85%E7%96%AB%E6%83%85" target="_blank">
       疫情
      </span>
      令出口订单锐减、工厂关闭，且各地封城措施，许多农民工因无法返工而工作不保；史丹福大学（Stanford University）一份调查显示，今年初中国各地相继封城，使农民工损失高达一千亿美元薪资。
      <br>
       <br>
        香港《南华早报》报导，中国官方公布失业数据势必低估实际状况，因仅计算4.4亿城镇劳工，排除失业或无法返工的农民工。
        <br>
         <br>
          香港望正资产管理公司（Upright Asset）首席经济学家刘陈杰报告推估，该疫情可能让2.05亿中国劳工、亦即逾25%就业人口落入“摩擦性失业”，就是希望工作，却无法上工或无工可上。
          <br/>
          <br/>
          他还指出，今年前两月，中国年营收超过两千万人民币的大型工业企业缩减五百万份工作，另小型工业公司流失两千万份工作，而工作损失最严重的是服务业，约消失1.8亿工作机会。
          <br/>
          <br/>
          中国社会安全系统几乎没有涵盖农民工，农民工沦为疫情最脆弱群体。
          <br/>
          <br/>
          中国西南财经大学调查发现，中国第一季年收入不满五万人民币的最穷困家庭，消费贷款需求指数较去年第四季的100跳升至110.5，但取得贷款的指数从上一季100下滑至90.5。
          <br/>
          <br/>
          中国人力资源与社会保障部承认，疫情对农民工的就业影响比其他群体更大、更直接；中国国务院4月21日宣布，未加入失业保险的农民工将首度可申请“低保”，并将重大基建项目的薪酬比率从10%提高至15%。
